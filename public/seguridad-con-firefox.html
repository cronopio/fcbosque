Vamos a enchular Firefox

Ahora si a lo BUENO

  • about:config
  • y escribimos lo siguiente en el buscador
  • browser.sessionstore.restore_on_demand = true
  • browser.sessionstore.restore_pinned_tabs_on_demand = true
  • network.http.pipelining = true
  • network.http.proxy.pipelining = true
  • network.http.pipelining.maxrequests = 8
  • security.dialog_enable_delay = 0
  • browser.urlbar.clickSelectsAll = True o false

Otras vainas mas :)

escribe el nombre de un sitio sin el “www” o el “.com”. Por ejemplo
“google” Luego pulsa Control+Enter

voala !!

y automáticamente añadirá el “www” y el “.com” y te llevará ahí como por arte de magia.
Para direcciones .net, pulsa Shift+Enter
para direcciones .org, pulsa Control+Shift+Enter.
Ctrl+Tab : Va a la siguiente pestaña. –>
Ctrl+Shift+Tab : Va a la anterior pestaña. <–
Ctrl+1-9 : Va al numero de pestaña que elijas.

Solución: CTRL + SHIFT + T
